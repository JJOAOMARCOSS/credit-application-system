## 🚀 Como Executar

### Usando o terminal:

```bash
# Clone o repositório
git clone git@github.com:JJOAOMARCOSS/request-credit-system.git

# Acesse a pasta do projeto
cd request-credit-system

# Execute o projeto
./gradlew bootRun
```
